Overview

iTweet defines a behaviour called itweep that represents a tweep, letting users implement it to use the Twitter Stream API with its credentials.

Usage

Just implement the functions defined by the itweep behaviour and then call itweep:start/3, itweep:start/4, itweep:start_link/3 or itweep:start_link/4 :)
